# Project Board – Collaborative Project Management Platform

## Overview
Project Board is a modern collaborative workspace designed to help teams manage projects, track tasks, communicate efficiently, and stay updated through real-time notifications.

## Features

### User Authentication
- User registration and login
- JWT authentication
- Persistent sessions

### Project Management
- Create projects
- Add descriptions
- Assign members

### Task Management
- Create and assign tasks
- Set priorities (Low, Medium, High)
- Add descriptions and due dates
- Update task statuses

### Kanban Workflow
- To Do
- In Progress
- Review
- Done

### Team Collaboration
- Comments on tasks
- Team member management
- Real-time updates

### Notifications
- Project updates
- Task changes
- Assignment alerts

## Technology Stack

### Frontend
- React 19
- Vite
- Socket.IO Client
- Lucide React
- CSS3

### Backend
- Node.js
- Express.js
- Socket.IO

### Security
- JWT
- bcryptjs

## Installation

```bash
npm install
npm run dev
```

Run backend only:

```bash
npm run server
```

Run frontend only:

```bash
npm run client
```
